MEK TYCOON - PROBABILITY SPHERE SELECTOR
=========================================

DESCRIPTION:
A 3D probability-based selection game where users drag and launch a sphere to land on different zones.
Each zone has different probabilities and rewards.

HOW TO RUN:
1. Open index.html in any modern web browser
2. That's it! No server required - completely standalone

HOW TO PLAY:
1. Click and drag anywhere on the sphere to build power
2. Release to launch the sphere
3. The sphere will spin and land on a probability zone
4. Different zones give different rewards (1-3 buffs)

FEATURES:
- 3 Rarity modes (Low, Medium, High) - click the RARITY button to cycle
- 4 Zone patterns (Digital Camo, Solid Zones, Organic, Lines)
- 3 Lighting modes (Standard, Neon Glow, Dark Mode)
- Real-time zone detection with visual feedback
- Probability-weighted distribution system

FILES:
- index.html (main game file)
- three.min.js (3D graphics library)
- README.txt (this file)

TECHNICAL:
- Built with Three.js for 3D rendering
- Fully self-contained, no external dependencies
- Works offline once loaded
